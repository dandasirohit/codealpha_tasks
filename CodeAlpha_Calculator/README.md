A simple **web-based calculator** built as part of CodeAlpha tasks.  
This project demonstrates basic frontend web development using **HTML**, **CSS**, and **JavaScript** to build a functional calculator UI.

Features:

✔️ Perform basic arithmetic operations:  
- Addition (+)  
- Subtraction (−)  
- Multiplication (×)  
- Division (÷)

✔️ Clear / reset display  
✔️ Support for decimal values  
✔️ Interactive UI  
✔️ Works with mouse 

This project was developed as part of CodeAlpha internship
